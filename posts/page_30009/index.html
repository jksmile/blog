定义：为其他对象提供一种代理以控制对这个对象的访问。

代理模式几个角色：
    1.主题接口：提取真正使用类与代理类的公共方法
    2.真正主题：真正处理请求的类
    3.代理类：真正主题的代理类

public ProxyNetWork implements Network{

    private BeijingNetwork bjNetwork;


public ProxyNetWork(){

        this.bjNetwork = new BeijingNetwork();
    }


    @override
    public void innerNet(){

        bjNetwork.innerNet();
    }

}
